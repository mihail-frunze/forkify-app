# Forkify App

A recipe search app built with JavaScript using a real API.

## Live Demo
https://forkify-mihaifrunze.netlify.app/

## GitHub
https://github.com/mihail-frunze/forkify-app

## How it works

- Search for recipes using a keyword
- Results are fetched from an external API
- Click a recipe to see full details
- Adjust servings and ingredients update automatically
- Bookmark your favourite recipes
- Add your own recipe
- Pagination is used for large result sets

## What I practised

- Working with APIs (fetch / async / await)
- Application architecture
- State management
- Modular JavaScript
- DOM updates and rendering
- Error handling
- Local storage (bookmarks)
